# Bilder

Hier gehört die Datei **`logo.png`** hinein — das runde Tourlocate-Logo.

Es wird verwendet als:
- Favicon (Browser-Tab) — siehe `<link rel="icon">` in `index.html`
- Apple-Touch-Icon (Homescreen auf iOS)
- Logo oben in der Seitenleiste (`.sb-brand img`)

Solange die Datei fehlt, blendet sich das Sidebar-Logo automatisch aus
(kein kaputtes Bild); der Schriftzug „Tourlocate" bleibt sichtbar.

Ein quadratisches PNG (z. B. 512×512 oder 1024×1024) genügt — der Browser
skaliert es für Favicon und Seitenleiste selbst.
